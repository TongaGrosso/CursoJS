Comandos para la consola de linux

pwd: Imprime la ubicación del directorio actual ls: lista los archivos o carpetas en la ubicación actual cd: desplaza entre diferentes directorios (cd ..) vuelva atrás cat: imprime el contenido de un archivo en la pantalla concadena archivos [OPTION] [FILE] cp: copia archivo en la segunda ubicación mv: mueve o renombra archivos mkdir: crea directorios rmdir: elimina directorios ln: vincula archivos y directorios sudo: otorga privilegios de administrador para realizar la acción touch: crea archivos rm: remueve/elimina archivos rm -d nombreDeLaCarpetaVacía: Borra carpeta vacía rm -rf nombreDeLaCarpeta: Borrado recursivo de carpeta locate: reboot: reinicia la pc o máquina virtual shutdown: apaga pc sudo apt install: instala programas Ctrl+C: Detiene la ejecución Ctrl+X: Cerrar editor de texto npm: instala paquetes de forma permanente npx: ejecuta programas/paquetes y luego los borra code: Abrir VSCode
